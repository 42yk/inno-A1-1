# 나만의 프롬프트 관리자

AI 작업에서 자주 사용하는 프롬프트를 제목, 내용, 카테고리, 즐겨찾기 상태로 관리하는 CLI 기반 Python 콘솔 프로그램입니다.

프롬프트 추가, 목록 조회, 카테고리별 조회, 검색, 상세 보기, 즐겨찾기 관리, Markdown 내보내기를 터미널 메뉴에서 실행할 수 있습니다.

## 제출 증빙

개발 환경 설정, 프로그램 실행 결과, Git 설정 확인용 캡처는 별도 문서에 정리했습니다.

- [제출 증빙 문서](EVIDENCE.md)

## 개발 및 실행 환경

- Python 3.10 이상
- 외부 패키지 설치 없음
- Python 표준 라이브러리 사용: `json`, `os`
- Git 기반 버전 관리

## 실행 방법

프로젝트 루트 디렉토리에서 다음 명령어를 실행합니다.

```bash
python3 prompt_manager.py
```

종료하려면 메인 메뉴에서 `0`을 입력합니다. 실행 중 `Ctrl+C` 또는 `Ctrl+D`를 입력해도 안내 메시지와 함께 안전하게 종료됩니다.

## 주요 기능

- 프롬프트 추가: 제목, 내용, 카테고리를 입력해 새 프롬프트를 등록합니다.
- 프롬프트 목록: 저장된 프롬프트를 등록순 또는 조회수 기준 인기순으로 확인합니다.
- 카테고리별 조회: 선택한 카테고리에 속한 프롬프트만 필터링합니다.
- 프롬프트 검색: 제목 또는 내용에 포함된 키워드로 프롬프트를 찾습니다.
- 상세 보기: 프롬프트 전체 내용, 카테고리, 즐겨찾기 상태, 조회수를 확인합니다.
- 수정 및 삭제: 상세 보기 화면에서 선택한 프롬프트를 수정하거나 삭제합니다.
- 즐겨찾기 관리: 자주 사용하는 프롬프트를 즐겨찾기로 등록하거나 해제합니다.
- Markdown 내보내기: 전체 프롬프트를 카테고리별 Markdown 파일로 내보냅니다.

## 파일 구성

메뉴 기능별로 파일을 분리하여 각 파일이 하나의 역할을 갖도록 구성했습니다.

```text
.
├── prompt_manager.py      # CLI 메뉴 출력, 메뉴 선택 라우팅, 종료 신호 처리
├── prompt_actions.py      # 프롬프트 추가, 목록, 검색, 상세 보기, 수정, 삭제
├── category_actions.py    # 카테고리 목록 구성 및 카테고리별 조회
├── favorite_actions.py    # 즐겨찾기 등록/해제 및 즐겨찾기 목록 조회
├── export_actions.py      # Markdown 내보내기 메뉴 처리
├── prompt_storage.py      # 기본 데이터, JSON 저장/불러오기, Markdown 파일 생성
├── README.md              # 프로젝트 소개 및 실행 안내
├── EVIDENCE.md            # 제출 증빙 캡처 정리
├── dev-setting/           # 개발 환경 설정 캡처
└── exec-images/           # 프로그램 실행 결과 캡처
```

## 데이터 저장

프로그램 실행 중 추가, 수정, 삭제한 프롬프트는 `prompts.json`에 저장됩니다. Markdown 내보내기를 실행하면 기본값으로 `prompts.md` 파일이 생성됩니다.

`prompts.json`과 `prompts.md`는 실행 중 생성되는 결과물이므로 Git 추적 대상에서 제외합니다.
