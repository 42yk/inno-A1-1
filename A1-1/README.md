# 📝 나만의 프롬프트 관리자 (Prompt Manager)

파편화되어 흩어져 있는 유용한 AI 프롬프트들을 체계적으로 관리하고, 필요할 때마다 검색 및 카테고리별로 조회하고 파일로 내보낼 수 있는 **CLI 기반의 파이썬 콘솔 프로그램**입니다.

## 🚀 주요 특징 및 기능

- **프롬프트 저장소**: 제목, 내용, 카테고리, 즐겨찾기, 조회수(views) 정보를 구조화된 파일(`prompts.json`)로 관리합니다.
- **자동 영속화 (JSON)**: 프롬프트 추가, 수정, 삭제, 즐겨찾기 토글, 상세 조회수 증가 시 즉각적으로 로컬 파일에 보관되어 재시작 시에도 정보가 유지됩니다.
- **정렬 옵션이 추가된 목록 조회**: 전체 프롬프트를 번호와 카테고리, 즐겨찾기 표시(`⭐`) 및 조회수 정보와 함께 보여줍니다. 목록 출력 시 **등록순**과 **인기순(조회수 기준)** 정렬 옵션을 제공합니다.
- **카테고리별 필터링 조회**: 기본 카테고리와 직접 추가한 카테고리 목록 중에서 원하는 항목만 필터링하여 프롬프트를 조회합니다.
- **상세 검색**: 프롬프트의 **제목** 또는 **내용** 속에 포함된 키워드로 일치하는 프롬프트를 일괄 검색합니다.
- **상세 보기 및 CRUD 고도화**: 특정 프롬프트를 선택해 내용을 전체 확인(이때 조회수 누적)하고, 바로 하단 서브 메뉴를 이용하여 해당 프롬프트를 **수정**하거나 **삭제**할 수 있습니다.
- **즐겨찾기 토글**: 자주 쓰는 프롬프트는 즐겨찾기로 등록(`⭐`)하고 즐겨찾기 전용 보관함에서 한눈에 모아볼 수 있습니다.
- **Markdown 파일 내보내기**: 전체 프롬프트 데이터를 카테고리별로 자동 분류하여 구조화된 아름다운 마크다운(`.md`) 보고서로 내보낼 수 있습니다.

### 실행 흐름 예시

1. **메인 메뉴 진입**:

   ```text
   === 나만의 프롬프트 관리 ===
   1. 프롬프트 추가
   2. 프롬프트 목록
   3. 카테고리별 조회
   4. 프롬프트 검색
   5. 프롬프트 상세 보기
   6. 즐겨찾기 관리
   7. 즐겨찾기 목록
   8. Markdown 내보내기
   0. 종료
   ===========================
   선택:
   ```

2. **프롬프트 목록 조회 시 정렬 옵션**:

   ```text
   --- 정렬 조건 선택 ---
   1. 등록순 (기본)
   2. 인기순 (조회수 기준)
   선택: 2
   ```

3. **Markdown 내보내기**:
   - 메뉴 8번 선택 시 저장할 파일명을 물어보며, 파일명을 입력하지 않고 엔터를 누를 시 기본값 `prompts.md`로 내보내집니다.

## 📂 지원하는 기본 카테고리

기본값으로 정의되어 제공하는 카테고리 목록은 아래와 같습니다. 프롬프트 추가 시 목록 번호를 선택해 지정하거나 직접 새로운 카테고리를 입력하여 확장할 수 있습니다.

1. **텍스트 생성** (예: 블로그 글, 광고 카피, 번역 등)
2. **이미지 생성** (예: 미드저니, DALL-E 프롬프트 등)
3. **영상 생성** (예: 광고 숏폼 콘티, 연출 지시문 등)
4. **페르소나** (예: IT 컨설턴트, 마케터 등 직업/성격 세팅)
5. **자동화** (예: 뉴스 요약 API 스크립트 템플릿 등)
6. **기타** (직접 지정)
